# Fake Job Prediction System

> NLP-Powered Fraudulent Job Posting Detection using TF-IDF and Streamlit.

## 🎯 Overview

- **Role:** Solo Developer
- **Timeline:** Nov 2025 - Jan 2026
- **Team:** Solo
- **Client:** Personal Project

## 🎯 The Challenge

Recruitment fraud has become increasingly common across online job portals, putting job seekers at risk of financial and identity theft. The challenge was to build an automated system capable of distinguishing between legitimate and fraudulent job advertisements based solely on their textual content with high accuracy and real-time performance.

## 💡 Solution

Developed an NLP-based machine learning application that preprocesses job descriptions using TF-IDF vectorization and classifies them using a Logistic Regression model. Deployed the solution as an interactive web application using Streamlit, allowing users to input job descriptions and receive instant predictions.

## 🛠️ Technologies Used

- **Languages:** Python
- **ML Libraries:** Scikit-learn, Pandas, NumPy
- **NLP Techniques:** TF-IDF Vectorization, Text Preprocessing, Tokenization
- **Deployment:** Streamlit, Streamlit Community Cloud

## 📊 Results & Impact

- **Real-Time Inference:** Achieved instant predictions through a user-friendly Streamlit interface.
- **Model Performance:** Evaluated using accuracy, precision, recall, and F1-score on unseen data to ensure robustness.
- **Accessibility:** Deployed live on Streamlit Community Cloud, making the tool accessible to anyone via a web browser.
- **Efficiency:** Built a lightweight inference pipeline with minimal dependencies for fast execution.
