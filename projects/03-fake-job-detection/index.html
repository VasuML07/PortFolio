Fake Job Detection System ● LIVE

NLP-powered application that detects fraudulent job postings in real-time using TF-IDF vectorization and Logistic Regression. Deployed on Streamlit Community Cloud for public access.

Role: Solo Developer Timeline: Nov 2025 - Jan 2026 Type: End-to-End ML Application

The Challenge

Recruitment fraud has become increasingly common across online job portals, putting job seekers at risk of financial loss and identity theft. The challenge was to build an automated system capable of distinguishing between legitimate and fraudulent job advertisements based solely on their textual content with high accuracy and real-time performance.

Key Features

📝 Text Classification

Binary classification of job postings as legitimate or fraudulent using supervised learning.

⚡ Real-Time Inference

Instant predictions through an interactive Streamlit interface accessible from any browser.

🧠 NLP Pipeline

TF-IDF feature extraction with text preprocessing, tokenization, and cleaning.

🌐 Live Deployment

Production-ready deployment on Streamlit Community Cloud with zero infrastructure management.

Technical Implementation

Built a complete end-to-end ML pipeline from raw text data to deployed web application.

  • Data Preprocessing: Lowercasing, special character removal, stopword filtering, and lemmatization.
  • Feature Engineering: TF-IDF vectorization to convert unstructured text into numerical features.
  • Model Training: Logistic Regression classifier optimized for binary classification with cross-validation.
  • Evaluation: Comprehensive metrics including accuracy, precision, recall, and F1-score on held-out test data.
  • Serialization: Model and vectorizer saved as Pickle files for efficient inference without retraining.
  • Web Interface: Streamlit app with input validation, prediction display, and confidence indicators.

Technologies Used

Python Scikit-learn TF-IDF Logistic Regression Streamlit Pickle NLP Text Classification
