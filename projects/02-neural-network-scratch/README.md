# Neural Network From Scratch

> Building Deep Learning One Matrix Multiplication at a Time using only NumPy.

## 🎯 Overview

- **Role:** Solo Developer
- **Timeline:** March 2026 - April 2026
- **Team:** Solo
- **Client:** Personal Project / Educational

## 🎯 The Challenge

Modern deep learning libraries like TensorFlow and PyTorch abstract away the underlying mathematics, making it difficult for learners to understand how neural networks actually "learn." The challenge was to rebuild a fully connected neural network from first principles to expose every computational step without relying on high-level APIs.

## 💡 Solution

Implemented a complete feed-forward neural network using only NumPy. Manually coded forward propagation, backpropagation via the chain rule, and the Adam optimizer. The project features a modular architecture with reusable dense layers and activation functions (ReLU, Softmax) to demonstrate the complete training pipeline.

## 🛠️ Technologies Used

- **Languages:** Python
- **Libraries:** NumPy (for all matrix operations and linear algebra)
- **Techniques:** Backpropagation, Gradient Descent, Cross-Entropy Loss, Adam Optimizer
- **Tools:** Git, Pickle (for model serialization)

## 📊 Results & Impact

- **Educational Value:** Created a transparent codebase that demystifies the internal mechanics of deep learning.
- **Modular Design:** Successfully implemented reusable layers and an experiment runner for easy model configuration.
- **Performance:** Achieved stable convergence on multi-class classification tasks using manual gradient computation.
- **Serialization:** Implemented model persistence using Pickle to save and reload best-performing weights.
